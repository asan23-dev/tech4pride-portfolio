HTML Glossary

element
A building block of html, made up of an opening tag, content and closing tag
attribute
extra information added inside an opening tag, like href or alt
semantic html
using tags that describe their meaning, like header, main, and footer
accessibility
designing so that everyone, including disabled users can use your page
void element
an element with no closing tag, like ig or meta
